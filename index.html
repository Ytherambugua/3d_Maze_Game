3D Maze Game

Experience a thrilling 3D maze adventure with raycasting and SDL2!

About the Project

This project involves creating a 3D maze game using raycasting. The game is built with SDL2 and C, providing a unique first-person experience as you navigate through the maze. The project covers various aspects of game development including rendering, collision detection, and user input handling.

Features

  • Real-time 3D rendering with raycasting
  • Player movement and rotation
  • Collision detection
  • Map parsing from files
  • Mini-map display
  • Textures and advanced graphical effects

Installation

  1. Clone the repository: git clone https://github.com/yourusername/3d-maze-game.git
  2. Navigate to the project directory: cd 3d-maze-game
  3. Install SDL2: sudo apt-get install libsdl2-dev
  4. Compile the project: make
  5. Run the game: ./3d-maze-game
